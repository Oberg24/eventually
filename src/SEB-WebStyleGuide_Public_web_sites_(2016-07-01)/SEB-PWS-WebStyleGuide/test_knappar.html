SEB PWS sites global default font-size is 16px, with a line-height of 1.4 unitless (40%) more than the font-size (22.4px). This is applied to the <body>.

Add the class .anfang-two-rows-intro, .anfang-three-rows-intro or .anfang-four-rows-intro on the wrapped character. If you want the anfang i outside the intro block, just change the last word of the class name. anfang-four-rows-intro to anfang-four-rows-body

Form elements

Here is the most common form element listed.

Inputs types and attributes Description
text Standard input text element, valid text
Som feedback text!
 text, class="error" Standard input text element, if not valid
password Standard input password element, hides the characters.
number ,pattern="[0-9]*" Standard input number element only 123456,.-+ caracters can be written.
email Standard input email element. Depending on browser support, the e-mail address can be automatically validated when submitted. Some smartphones recognize the email type, and adds ".com" to the keyboard to match email input.
radio Radio select element. Created only with css. The trick is to map the label tag binding the radio button id with the attribute for="radioButtonId" together. Combining them and you can have any look you want :-)
checkbox Checkbox element. Created only with css and using the check indecator from the SEB font icon library. The trick is to map the label tag binding the checkbox id with the attribute for="checkboxId" together. Combining them and you can have any look you want :-)
Select element attributes Description
class Select something from the dropdown list.
Som feedback text!
 class="error" The select element look if the validation is invalid.
Textarea element attributes Description
class Textarea element. The textarea element is uset to send plain texts.
Som feedback text!
 class="error" Textarea element. The textarea element having invalid text.

Buttons

Button styles can be applied to anything with the .btn class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

Button class="" Description
btn simple Standard gray button with gradient
btn wanted Provides extra visual weight and identifies the primary action in a set of buttons
